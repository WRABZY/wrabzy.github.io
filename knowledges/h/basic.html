Базовые конструкции

Импорт

import Module.Name - импортирует модуль

Функция

simpleFunction x = 1 - принимает аргумент и возвращает единицу

if then else

isOdd x = if x `mod` 2 == 1 then True else False - ветвь else обязательна, для инфиксного применения функций, они помещаются в между обратными кавычками ` `

let и where

let a = b + 1 in <выражение> - a заменяется на b + 1 в следующем <выражении>
where a = b + 1 - a заменяется на b + 1 в предыдущем блоке

Лямбды

\x -> x + 1 - простая лямбда-функция

Доступ к элементу списка по индексу

"FUTURAMA" !! 2 - символ 'T'

Сопоставление с образцом

function list@(x:xs) - list - список, x - его голова, xs - его хвост, _ - универсальный образец

Левая свёртка foldl

foldl fun init list - лениво сворачивает список слева, foldl' - не ленивая, вместо инициализирующего значения первый элемент

Правая свёртка foldr

foldr fun init list - сворачивает список (возможно бесконечный) справа более эффективно, чем foldl, foldr' - вместо инициализирующего значения последний элемент
