Тип Optional<T>

Объект типа Optional<T> является оболочкой значения типа T или null.

Создание объектов типа Optional<T>

  • Optional<T> optionalValue = Optional.of(value) - если value == null throws NullPointerException
  • Optional<T> optionalValue = Optional.empty()
  • Optional<T> optionalValue = Optional.ofNullable(value) - если value == null, возвращает Optional.empty()

Примеры использования

  • T result = optionalValue.orElse(defaultValue);
  • T result = optionalValue.orElseGet(supplier);
  • T result = optionalValue.orElseThrow(supplier);
  • optionalValue.ifPresent(consumer);
  • Optional<T> newOptional = optionalValue.map(function);
  • Optional<T> newOptional = value.f().flatMap(T::g) - если f() возвращает объект, обёрнутый в Optional<T>, то flatMap позволяет извлечь объект из обертки, применить его метод g() и поместить результат в обертку.
