Работа с базами данных

Термины и сокращения

JDBC

Java Database Connectivity Java API для работы с базами данных.
  1. Connection - соединение с БД и режим работы
  2. Statement - выражение обращения к базе
  3. ResultSet - объект с результатом запроса

ORM

Object Relational Mapping Отображение объектов на отношения. Основные аннотации:
  • @Entity - объект класса можно "переложить" в таблицу
  • @Table - связывает класс и таблицу
  • @Id - поле является первичным ключом в таблице
  • @Column - связывает поле и колонку в таблице
Hibernate - пример ORM.

JPA

Java Persistency API JDK API для ORM

Hibernate

Популярная библиотека для ORM, implements JPA

DAO

Data Access Object Объект, позволяющий получить доступ к данным. Центральный объект ORM. Преобразует объекты в таблицы, а таблицы в объекты.

DataSet

Объект, содержащий данные одной строки таблицы. В терминах JPA DataSet - это Entity.
