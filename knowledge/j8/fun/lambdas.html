Лямбды

Лямбдой можно воспользоваться, когда ожидается объект функционального интерфейса. Ссылка на лямбду должна иметь тип функционального интерфейса. Например, присваивание ссылке типа Object лямбды вызовет ошибку компиляции.
Однострочная запись с параметрами (String first, String second) -> first.length() - second.length()
Многострочная запись с параметрами (String first, String second) -> {   if (first.length() < second.length()) return -1;   else if (first.length() > second.length()) return 1;   else return 0; При многострочной записи необходимо указание return.
Однострочная запись без параметров () -> { for (int i = 100; i >= 0; i--) System.out.println(i); }
Если типы параметров могут быть выведены, объявление можно опустить: Comparator<String> comp = (first, second) -> first.length() - second.length()
Если у метода только один параметр, и его тип можно вывести: ActionListener listener = event -> System.out.println("The time is " + new Date());
