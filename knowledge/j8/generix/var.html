Вариантность

Коваринатность - сохранение иерархии наследования исходных типов в производных типах в том же порядке. В java ковариантны:
  • Массивы String[] strings = new String[] {"a", "b", "c"}; Object[] arr = strings;
  • Дженерики с "? extends" List<Integer> ints = new ArrayList<Integer>(); List<? extends Number> nums = ints;
Контравариантность - обращение иерархии исходных типов на противоположную в производных типах. В java контравариантны:
  • Дженерики с "? super" List<Number> nums = new ArrayList<Number>(); List<? super Integer> ints = nums;
    • Инвариантность - отсутствие наследования между производными типами. В java инвариантны:
    • Дженерики без wildcard List<Integer> ints = Arrays.asList(1,2,3); List<Number> nums = ints; // compile-time error
