int hashCode()
Не переопределенный метод hashCode() вычисляет хеш-код по адресу памяти, занимаемой объектом.
Метод Objects.hashCode() безопасно обрабатывает null-значения.
Метод Double.hashCode(double value) поможет обойтись без создания объекта Double (относится ко всем обёрткам).
Метод Objects.hash(o1, o2, o3) вычислит хеши и объединит их в один хеш-код.
Метод Arrays.hashCode() вычислит хеши элементов массива и объединит их в один хеш-код.
