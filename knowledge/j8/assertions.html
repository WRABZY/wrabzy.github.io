Assertions

assert условие assert условие : сообщение Если условие не выполняется, то генерируется исключение AssertionError
По умолчанию утверждения запрещены. Разрешить их можно, указав -enableassertions (-ea) при запуске программы: java -enableassertions App
Разрешать утверждения можно в отдельных классах или целых пакетах: java -ea:MyClass -ea:com.mycompany.mylib ... MyApp Эта команда разрешает утверждения в классе MyClass, а также во всех классах пакета com.mycompany.mylib и его подчиненных пакетов.
Параметр -ea... разрешает утверждения во всех классах из пакета по умолчанию, а параметр -da (-disableassertions) запрещает утверждения в отдельных классах и пакетах: java -ea:... -da:MyClass MyApp
