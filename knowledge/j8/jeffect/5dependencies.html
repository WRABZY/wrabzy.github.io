5. Зависимости

Внедрение зависимостей предпочтительней, чем hardcoded resources.

Суть

Требуемые ресурсы стоит передавать конструктору при создании экземпляра. Это одна из форм внедрения зависимостей (dependency injection).

Внедрение зависимостей может засорять проекты, содержащие тысячи зависимостей. Справиться с этим помогают фреймворки внедрения зависимостей: Dagger, Guice, Spring. API, разработанные для внедрения зависимостей вручную, тривиально адаптируются для использования этими фреймворками.
