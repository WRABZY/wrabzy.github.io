Интерфейсы

Все методы интерфейса являются открытыми без указания модификатора public. Однако при реализации интерфейса модификатор public должен быть указан.
В интерфейсах отсутствуют поля экземпляра. Объявление double SPEED_LIMIT = 95; внутри интерфейса означает public static final double SPEED_LIMIT. Класс, реализующий интерфейс автоматически наследует константы.
В интерфейсах могут располагаться статические методы.
Для любого интерфейсного метода можно предоставить реализацию по умолчанию. default int compareTo(T other) { return 0; }

Если метод реализован по умолчанию в нескольких сущностях:

В приоритетах выигрывает суперкласс. При отсутствии метода в суперклассе, но при наличии в нескольких суперинтерфейсах - требуется переопределение метода. Переопределение происходит следующим образом: public String getName() { return Nameable.super.getName(); } Это касается и ситуаций, когда только один из интерфейсов предоставляет одноименный метод по умолчанию.
