Примеры кода

Создание логера private static final Logger logger = Logger.getLogger("name.of.logger");
Задание уровня логеру logger.setLevel(Level.FINE);
Вызовы методов логирования logger.warning(message); logger.log(Level.WARNING, message); logger.logp(Level.WARNING, "ClassName", "methodName", message); logger.entering("ClassName", "methodName"); logger.entering("ClassName", "methodName", new Object[]{param1, param2, param3}); logger.exiting("ClassName", "methodName"); logger.exiting("ClassName", "methodName", result);
Логирование исключений logger.throwing("ClassName", "methodName", throwable); При вызове метода throwing() запись имеет уровень FINER. logger.log(Level.WARNING, "message", throwable);
Установка своего хендлера Logger logger = Logger.getLogger("com.mycompany.myapp"); logger.setLevel(Level.FINE); logger.setUseParentHandlers(false); Handler handler = new ConsoleHandler(); handler.setLevel(Level.FINE); logger.addHandler(handler);
