Символьныe

char = character(n)    # дополняется пробелами до длины n text varchar = character varying(n)
Для экранирования кавычки ' она удваивается. Вместо кавычек для обозначения символа может использоваться удвоенный $, тогда экранирование кавычек не потребуется.
Указание E перед первой кавычкой позволяет использовать строковые константы из C. SELECT E'PGDAY\n17'    # тут перевод строки в виде \n При этом для вставки обратного слэша, его нужно экранировать как \\, а для вставки кавычки либо \', либо ''.

Числовые

Целочисленные

  • smallint = int2
  • integer = int4
  • bigint = int8

Фиксированной точности

  • numeric
  • decimal
Scale (масштаб) показывает число значащих цифр после запятой. Precision (точность) указывает общее число цифр. Пример: у числа 12.3456 масштаб 4 и точность 6. numeric(prec, scale) numeric(6, 4)

С плавающей точкой

  • real = float(p), если значение в диапазоне от 1 до 24
  • double precision = float(p), если значение в диапазоне от 25 до 53
