Преобразование в постфикс

Преобразование инфиксных выражений в обратную польскую запись

Если из входной строки считан { ЭЛЕМЕНТ }, то выполнить действие
  • { ОПЕРАНД } Записать в выходную (постфиксную) строку
  • { Открывающая круглая скобка ( } Занести в стек
  • { Закрывающая круглая скобка ) } Пока в стеке остаются элементы: - извлечь элемент - если он отличен от (, записать в выходную строку При чтении элемента ) выйти из строки
  • { Оператор (opThis) } Если стек пуст, - занести оператор в стек Иначе - пока в стеке остаются элементы   - извлечь элемент   - Если это (, то занести его в стек   - Если это оператор (opTop), то     - Если opTop < opThis, то занести в стек opTop     - Если opTop >= opThis, то занести opTop в выходную строку - если opTop < opThis или при извлечении ( выйти из цикла - занести в стек opThis
  • { Конец входных данных } Пока в стеке остаются элементы: извлечь элемент в выходную строку.
