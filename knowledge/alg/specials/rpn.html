Преобразование в постфикс

Преобразование инфиксных выражений в обратную польскую запись

Если из входной строки считан { ЭЛЕМЕНТ }, то выполнить действие
  • { ОПЕРАНД } Записать в выходную (постфиксную) строку
  • { Открывающая круглая скобка ( } Занести в стек
  • { Закрывающая круглая скобка ) } Пока в стеке остаются элементы: - извлечь элемент - если он отличен от (, записать в выходную строку - при чтении ( - выйти из цикла
  • { Оператор (opThis) } Пока в стеке остаются элементы:   - извлечь элемент   - Если это (, то занести его в стек и выйти из цикла   - Если это оператор (opTop), то     - Если opTop < opThis, то занести в стек opTop и выйти из цикла     - Если opTop >= opThis, то занести opTop в выходную строку Занести в стек opThis
  • { Конец входных данных } Пока в стеке остаются элементы: извлечь элемент в выходную строку.
