Стеки, деки и очереди

Стеки, деки и очереди являются абстрактными типами данных, то есть определяют интерфейс доступа к данным, но не внутреннее устройство структуры данных.

Стек (Last In First Out - LIFO)

В стеке доступен только тот элемент, который был помещен в него последним. После его извлечения из стека пользователю становится доступен предпоследний и т. д.
Основные операции, которые предоставляет стек пользователю:
  • void push(T element) - вставка элемента на вершину стека
  • T pop() - извлечение последнего элемента
  • T peek() - чтение последнего элемента без его извлечения из стека
  • boolean isEmpty() - проверка, пуст ли стек
  • boolean isFull() - проверка, полон ли стек

Очередь (First In First Out - FIFO)

В очереди доступен только тот элемент, который был помещен в нее раньше остальных. После его извлечения из очереди пользователю становится доступен следующий и т. д.
Основные операции, которые предоставляет очередь пользователю:
  • void insert(T element) - вставка элемента в конец очереди
  • T remove() - извлечение элемента, находящегося в начале очереди
  • T peek() - чтение элемента, находящегося в начале очереди, без его извлечения из очереди
  • T rearPeek() - иногда доступно чтение элемента, находящегося в конце очереди, без его извлечения из очереди
  • boolean isEmpty() - проверка, пуста ли очередь
  • boolean isFull() - проверка, полна ли очередь

Приоритетная очередь

В приоритетной очереди элементы упорядочиваются по ключу. Новые элементы вставляются в позициях, сохраняющих порядок сортировки. Приоритетная очередь должна обеспечивать относительно быструю вставку. Поэтому приоритетные очереди часто реализуются на основе кучи.

Дек

Вставка элементов в дек и удаление элементов из дека могут производиться с обоих концов.
Основные операции, которые предоставляет дек пользователю:
  • void insertLeft(T element) - вставка элемента в дек слева
  • void insertRight(T element) - вставка элемента в дек справа
  • T removeLeft() - извлечение элемента, находящегося в деке слева
  • T removeRight() - извлечение элемента, находящегося в деке справа
и т. д.
