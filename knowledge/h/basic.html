let и where

let a = b + 1 in <выражение>a заменяется на b + 1 в следующем после "in" <выражении>
where a = b + 1a заменяется на b + 1 в блоке, предшествующем "where".
"where" должно отделяться отступами относительно блока, к которому оно относится.

Сопоставление с образцом

function list@(x:xs)Здесь: list - весь список, переданный в функцию, x - его голова, xs - его хвост.
_ (символ нижнего подчеркивания) - универсальный образец, с которым совпадут любые данные, переданные в функцию

Левая свёртка | foldl

Левая свёртка не может использоваться с бесконечными списками foldl fun init listЛениво сворачивает список слева, применяя fun к кажому элементу списка. Первым используется инициализирующее значение init.
foldl' - неленивая версия foldl, которая вместо инициализирующего значения использует первый элемент списка.

Правая свёртка | foldr

foldr fun init listCворачивает список (возможно бесконечный) справа. Более эффективна, чем foldl.
foldr' - неленивая версия foldr, которая вместо инициализирующего значения использует первый элемент списка.
