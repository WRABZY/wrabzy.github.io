Do - нотация предназначена для работы внутри контекста IO
Переменные, значения которым были присвоены с помощью <-, позволяют работать с типом IO a, будто это значение типа a. Выражения let используются при создании переменных, не принадлежащих типу IO. Например: name <- getLine     --getLine возвращает IO String let statement = helloPerson name     --helloPerson принимает String --так как значение name присвоено с использованием <-, ее можно использовать как тип String
